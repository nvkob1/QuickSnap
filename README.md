# Window Screenshot Taker

A simple, easy-to-use application for capturing screenshots of specific windows on your computer.

## Features

- Select any open window from a dropdown list
- Capture screenshots with a single click
- Save screenshots as PNG files
- Minimizable window for easy multitasking

## How to Use

1. Launch the application
2. Choose a window from the dropdown list
3. Click "Capture Screenshot" 
4. Select where to save your screenshot
5. Done!

## Requirements

- Windows operating system
- .NET Framework 4.6.1 or higher

## Installation

Simply download and run the installer. No additional software required.

## Tips

- Use the "Refresh Window List" button if you don't see a recently opened window
- You can minimize the app while setting up other windows for capture
- Screenshots are saved as high-quality PNG files
